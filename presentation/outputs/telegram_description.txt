Собрали локальный интерактивный прототип платежной платформы на Next.js, TypeScript, Prisma и SQLite.

Внутри есть роли, дашборды, ордера, выплаты, реквизиты, балансы, комиссии, апелляции, уведомления, журнал событий, API-демо и кликабельные сценарии.

Это не статичный макет: статусы меняются, балансы пересчитываются, апелляции обрабатываются, события фиксируются в журнале. Прототип можно запустить локально и показать партнеру, инвестору или команде, чтобы быстро объяснить бизнес-логику продукта.

Запуск: npm install → npx prisma migrate dev → npm run seed → npm run dev
